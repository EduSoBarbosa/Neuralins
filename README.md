# Neuralins - Cursos e Mentorias

Projeto Integrador do curso superior de Desenvolvimento de Software Multiplataforma (DSM) - FATEC Luigi Papaiz.

## Sobre o projeto

Neuralins é uma plataforma que ajuda jovens através de teste vocacional, cursos e mentorias a se identificar com alguma profissão!

O projeto tem como meta direcionar os que estão no processo de entrar no mercado de trabalho, sem saber por onde começar.

Nosso projeto tem como base os Objetivos de Desenvolvimento Sustentável (ODS) da ONU:

**ODS 4 -** Educação de Qualidade

**ODS 8 –** Trabalho decente e crescimento econômico
